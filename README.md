
# Huffman Archiver
Console application for lossless file compression and decompression using Huffman coding.


## What the program does

- Compresses files using Huffman coding
- Decompresses previously compressed files
- Stores frequency table inside the compressed file
- Uses bit-level I/O for writing and reading encoded data
- Includes unit and integration tests

```
data/

```
## How it works
- The program reads the input file and counts symbol frequencies.
- A Huffman tree is built using a priority queue.
- Prefix codes are generated by traversing the tree.
- The input data is encoded into a bit sequence.
- The compressed file stores:
   - number of unique symbols
   - frequency table
   - encoded bit length
   - encoded data
- During decompression, the tree is rebuilt from the stored frequencies and the original file is restored.
## Build instructions
```
mkdir build
cd build
cmake ..
cmake --build .
```
## Usage

```
./huffman_archiver compress <input_file> <output_file>
./huffman_archiver decompress <input_file> <output_file>
```
Example:
```
./huffman_archiver compress input.txt compressed.bin
./huffman_archiver decompress compressed.bin output.txt
```
